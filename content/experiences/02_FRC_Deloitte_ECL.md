## Context

The client was in the process of merging with another bank, and needed to compute the sensitivity of ECL to different internal and external macro-economic scenarios for their first joint annual report.

## Contribution

- Helped with coordinating and de-risking the delivery using agile tools (project management platform and code repository for versioning critical code).
- Created a python automation library for the web UI-based R-based tool used to compute the ECL sensitivities (automation was required for sensitivity computation).
- Engaged responsible stakeholders for clarifying appropriate risk segment mappings, required for reporting.

## Outcome

Code versioning greatly improved accountability and transparency in ECL computation delivery.
