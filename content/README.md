# Charilaos Mylonas profile configuration

This folder follows the structure of the supplied example:

- `profile.json` contains profile-level metadata and the ordered list of experience IDs.
- `experiences/` contains one Markdown file per public project or research contribution.
- `assets/` contains self-contained SVG illustrations referenced by the experience files.
- `SOURCE_NOTES.md` records the public evidence and wording decisions used to prepare the profile.

The project set intentionally includes mature open-source software, personal prototypes, and completed peer-reviewed research. The drug–target interaction item is retained because it is featured on the portfolio, but its work-in-progress status is stated explicitly.
